## Overview

The Admin Dashboard is your control center for managing all content on the Nandi Youth Assembly website. Access it anytime by visiting `/admin` on your website.

## Getting Started

### Accessing the Admin Dashboard

1. Navigate to your website: `nyassembly-wdk2vuhb.manus.space`
2. Click the **Admin** link in the top navigation menu
3. You'll be taken to the Admin Dashboard

### Dashboard Layout

The dashboard is organized into three main sections:

- **Hero Images** — Manage carousel images on the homepage
- **Leadership Team** — Update team member profiles
- **Site Settings** — Configure contact information and social media

## Deployment & Export

### Current Domain

Your website is hosted at: **nyassembly-wdk2vuhb.manus.space**

### Exporting Your Website

To download your complete website for backup or external hosting:

1. Go to the **Management UI** (click the menu icon)
2. Click the **⋯ (More)** button
3. Select **Download as ZIP**
4. Extract the ZIP file to your computer

### Using a Custom Domain

To use your own domain (e.g., nandiyouthassembly.org):

1. Go to **Management UI** → **Settings** → **Domains**
2. Click **Add Custom Domain**
3. Enter your domain name
4. Follow the DNS setup instructions
5. Your site will be available at your custom domain

### Deploying to External Hosting

See **DEPLOYMENT_GUIDE.md** for detailed instructions on deploying to:
- Vercel

## Best Practices

### Content Management

1. **Keep Information Current** — Update team member information when roles change
2. **Use High-Quality Images** — Professional photos make a better impression
3. **Write Clear Bios** — Help visitors understand each team member's role
4. **Update Social Links** — Ensure all social media URLs are correct and active

### Website Maintenance

1. **Regular Backups** — Download your website regularly as a backup
2. **Test Changes** — Preview changes before publishing
3. **Mobile Check** — Test how your website looks on phones and tablets
4. **Link Verification** — Periodically check that all links work correctly

### Accessibility

1. **Alt Text** — All images should have descriptive titles
2. **Clear Text** — Use simple, clear language
3. **Readable Fonts** — Ensure text is large enough to read
4. **Color Contrast** — Make sure text stands out from backgrounds

## Troubleshooting

### Admin Dashboard Not Loading

**Problem:** The admin page shows "Access Denied"  
**Solution:** Only admin users can access the dashboard. Contact the site owner to request admin privileges.

### Changes Not Saving

**Problem:** I updated information but it's not showing on the website  
**Solution:** Full backend integration is coming soon. Currently, the admin dashboard is in preview mode. Changes will be saved automatically once backend integration is complete.

### Images Not Uploading

**Problem:** I can't upload new images  
**Solution:** Image upload functionality is coming soon. Currently, you can only view and manage existing images.
